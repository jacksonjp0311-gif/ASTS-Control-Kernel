# ASTS — Autonomous Stability & Tuning System

## Overview
ASTS is a layered execution control kernel for drift monitoring and stabilization.

## Architecture
Execution → Telemetry → Assessment → Constraints → Stabilizer → Output

## Features
- Multi-scale drift tracking
- Cooldown reset logic
- Adaptive execution gating
- Stability-first control
